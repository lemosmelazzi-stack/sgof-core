Regla simple (importantísima)

👉 Siempre que creás algo:

INSERT INTO tabla...

👉 después hacé:

SELECT * FROM tabla;
💡 Ejemplo real tuyo

Vos hiciste:

INSERT cliente

después:

SELECT * FROM clientes;

✔ perfecto

🚀 Nivel siguiente (pro, pero simple)

En vez de ver todo:

SELECT * FROM viajes;

podés ver solo lo importante:

SELECT codigo, estado, importe_final
FROM viajes;
🎯 Resumen corto

SELECT * FROM tabla; → ver todo

SELECT columna1, columna2 FROM tabla; → ver específico
